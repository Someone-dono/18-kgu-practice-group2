Цели и задачи.
Получение практики с работой Hibernate.

Индивидуальное задание.
1) Создать таблицы БД на SQL server.
2) Ознакомиться с фреймворком Hibernate.
3) Подключить таблицы БД к среде разработки intellij IDEA на java
с помощью фреймворка Hibernate.
4) Создать связи между таблицами в среде разработки intellij IDEA на java
с помощью mapping.
5) Получить данные из таблиц, составленных в БД.

Теоретическая и практическая части.
теоретическая:
базовая информация hibernate плюсы и минусы.
найти фреймворки с аналогичным функционалом и сравнить.

практическая:
описать разработку таблиц бд и работу с hibernate.
